# 의약품 정보 웹사이트 (Medicine Recommend)

증상에 따라 간편하게 알맞은 의약품을 추천합니다.

## 기능

- **의약품 검색**: OpenFDA API 활용 (한글/영문 지원)
- **상세 정보**: 효능·효과, 용법·용량, 주의사항, 부작용, 금기, 약물 상호작용
- **약물 상호작용 검사**: 복용 중인 약 2개 이상 입력 시 상호작용 검사
- **알약 식별**: 모양, 색상, 각인으로 알약 검색
- **내 복용약**: 저장, 알레르기·상호작용 검사 연동

## 실행

- **GitHub Pages**: https://psyeong.github.io/medicine_recommend/
- **로컬**: `python -m http.server 8080` 후 http://localhost:8080

## 주의사항

본 정보는 참고용이며, 의료 상담을 대체하지 않습니다.
